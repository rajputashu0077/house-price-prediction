# House Price Prediction using Linear Regression

## Overview
This project predicts house prices using a Machine Learning model built with Linear Regression in Scikit-learn.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Data preprocessing
- Train-test split
- Linear Regression model
- Model evaluation using MAE and R² score
- House price prediction

## Model Performance
- Mean Absolute Error (MAE): 3.19
- R² Score: 0.67

## How it Works
1. Load housing dataset
2. Split data into training and testing sets
3. Train Linear Regression model
4. Predict house prices
5. Evaluate model performance

## Output Example
Predicted House Price: 29.0  
Actual House Price: 23.6
